## Features

This API contains the following functionality

  <Card title="Register Studios" icon="pen-to-square" href="https://mintlify.com/docs/quickstart">

  </Card>
  <Card title="Enter as Players" icon="image" href="https://mintlify.com/docs/development">
    Players are the user accounts that can interact with the studio products and manage communities with matches.
  </Card>
  <Card title="Get Achievements" icon="sparkles">
    A detail about the games that is awarded upon completion of a specific task.
  </Card>
  <Card title="Participate in Matches" icon="sparkles">
    An online competition to face other players.
  </Card>
  <Card title="Show Glory on Leaderboards" icon="sparkles">
    Place where the information about matches of general in-game progress is shown.
  </Card>
